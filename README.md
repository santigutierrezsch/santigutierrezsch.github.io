## 🚀 Setup & Installation

Clone the repo:

```bash
git clone https://github.com/santigutierrezsch/santigutierrezsch.github.io.git
cd santigutierrezsch.github.io
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
